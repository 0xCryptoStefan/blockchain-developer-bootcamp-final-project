NFT Public Notes App

This is a simple Decentralised Application (dApp), developed for the ConsenSys Blockchain Developer Bootcamp final project by Stefan. It makes use of NFTs to store messages on the blockchain and allows users to retrieve and display all committed messages (NFTs).
Please connect to the Kovan testnet to use this dApp.
